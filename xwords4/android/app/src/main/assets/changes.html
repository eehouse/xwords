CrossWords 4.4.158 release

This release fixes stalls that could occur if you were using certain Phonies settings.

Please take this survey to help determine what gets done next in CrossWords.

New with this release

  • Fix timing problems when the "Phonies" setting is DISALLOW
  • Clean up "Edit player" dialog
  • Add support for Hungarian wordlists
  • Fix a rare crash closing wordlists
  • Include new Turkish translation (via Weblate)

  • Support duplicate-style play (popular in France)
  • Improve play-by-data-sms workaround using NBSProxy
